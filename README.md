# pattern-recognition

1. Activations: Collection of all relevant and useful Activation functions along with their derivatives

2. CNN-layers: Implementation of some CNN layers which include pooling, convolution, batch-normalization. Also contains api to calculate output layer output size

3. DeltaLearning: Implementation of Delta learning algorithm both sequential and batch

4. k-nearest-neighbor: Simple implementation of KNN

5. Linear-disciminant-function: Implemenation of Linear disciminant function alogrithm

6. MLP-1: Implemenation of forward-propagation, backpropagation, weight-update APIs over 1 epoch of input data in Multi layer perceptron

7. NegativeFeedbackLearning: Implementation of basic as wells as Regulatory feedback(also called Decisive Input Modulation) learning

8. Widrow-hoff-sequential: Implementation of widrow-hoff algorithm with sequential learning
